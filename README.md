# IGF2BP2 Project

Bioinformatics and transcriptomics analysis pipeline for IGF2BP2-related studies.

## Contents

- Python-based network and correlation analysis
- R-based RNA-seq and WGCNA workflows
- Circadian dataset visualization
- Figure generation notebooks

## Environment Setup

### Python

```bash
conda env create -f python_environment.yml
conda activate igf2bp2-python
```

### R

```bash
conda env create -f r_environment.yml
conda activate igf2bp2-r
```
